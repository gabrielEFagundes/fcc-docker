## About the project

This is a port for [Free "Claude Code"](https://github.com/Alishahryar1/free-claude-code) to run on a Docker container, so you don't have to worry about downloading everything manually on your machine.

## Features

This project contains all the features from *fcc*, including:
- Full reading access to your user root files (`/workspace/**`), such as Documents and Downloads
- Full editing and creating permissions, if you allow him.
- Undestands `CLAUDE.md` / agentic files.
- Pretty much anything that *fcc* has to give.
